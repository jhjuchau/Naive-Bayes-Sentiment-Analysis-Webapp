Motivation

We wanted to simplify the overall process of sentiment analysis for the common person. Not only does our web application simplify the search process, but it provides a visual of the results easy enough for anyone to comprehend. By making the project as user friendly as possible, it allows our web application to be commercially useful. From a common user that wants to how others feel about their favorite basketball player to a hopeful presidential candidate, we provide a source that would show the general consensus of how a topic is perceived. Not only does this application help normal users, but it also benefits public relation teams that need to know how the population feels about their industry. By using the data that has been provided, these teams can make any changes that would make their industry more appeasing to the masses.
